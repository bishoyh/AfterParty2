Introduction

This tutorial will take you through the steps required to upload an annotate an existing transcriptome assembly using AfterParty. We will assume that you have been given a user account on a live AfterParty instance. For the purposes of the tutorial, we will be using a small assembly of 1000 reads from the transcriptome of a nematode, L. sigmodontis, which can be downloaded https://raw.github.com/mojones/AfterParty2/master/sample_data/smallAssembly_out.ace.
